
# 📊 Bank Loan Analysis Dashboard

## 🚀 Project Overview

This project focuses on analyzing bank loan data using Power BI and SQL to generate meaningful insights about lending performance, customer behavior, and risk assessment.

The dashboard helps in monitoring loan distribution, identifying good and bad loans, and understanding key financial metrics.

---

## 🛠 Tools & Technologies Used

* Power BI (Dashboard & Visualization)
* SQL (Data Analysis & KPI Calculation)
* Excel (Data Cleaning)

---

## 📊 Key KPIs

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)

---

## ✅ Good Loan Analysis

* Loans with status **Fully Paid / Current**
* Good Loan Percentage
* Good Loan Funded Amount
* Good Loan Received Amount

---

## ❌ Bad Loan Analysis

* Loans with status **Charged Off**
* Bad Loan Percentage
* Bad Loan Funded Amount
* Bad Loan Received Amount

---

## 📈 Dashboard Insights

* Monthly Loan Trends
* State-wise Loan Distribution
* Loan Term Analysis
* Employee Length Impact
* Loan Purpose Breakdown
* Home Ownership Analysis
